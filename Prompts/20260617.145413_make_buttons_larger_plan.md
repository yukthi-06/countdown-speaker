# Make Buttons and Text Larger Plan

We will increase the visual and textual size of the timer control buttons, remove parent layout weight constraints to allow buttons to dynamically occupy 50% width when others are hidden, and decrease the timer digits from 64sp to 48sp to prevent line wrapping issues.
