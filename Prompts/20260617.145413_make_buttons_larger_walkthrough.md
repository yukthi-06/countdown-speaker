# Make Buttons and Text Larger Walkthrough

We increased button sizing and text visibility while scaling down the clock layout to prevent digits from wrapping.

## Changes Made

### Layouts
- Decreased timer text size from 64sp to 48sp in `fragment_countdown.xml` and `fragment_countup.xml`.
- Removed `android:weightSum` constraints from button controls so they take up 50% width when alternate views are `GONE`.
- Increased button padding and changed text size from 11sp to 16sp.
