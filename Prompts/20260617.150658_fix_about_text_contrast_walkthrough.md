# Fix About Text Contrast Walkthrough

We resolved the text contrast issues on the About screen under dark mode profiles by replacing static color references with theme text parameters.

## Changes Made

### Layouts
- Updated `activity_about.xml` to replace static `@color/text_primary` with themed `?android:attr/textColorPrimary` and static `@color/text_secondary` with themed `?android:attr/textColorSecondary`.
- Updated divider and card boundary strokes to draw using `?attr/colorOutline` instead of `@color/divider`.

## Verification
- Confirmed that developer details and build information text displays in clearly legible styles under both theme setups.
