# User Prompt

Create a complete Android application in **Java (not Kotlin)** named **SpeakTimer**.

## Objective

Build an Android app that provides both:

1. **Count Down Timer**
2. **Count Up Timer**

using a **TabLayout + ViewPager2** interface with two tabs:

* Count Down
* Count Up

The app must be production-quality, compile successfully, and be compatible with GitHub Actions CI builds.

---

# Technical Requirements

## Language & Framework

* Java only
* AndroidX
* Material Design Components
* Minimum SDK: 24
* Target SDK: latest stable Android SDK
* Compile SDK: latest stable Android SDK
* Gradle: latest version currently supported by GitHub Actions Android builds
* Use Gradle Wrapper
* Use View Binding
* No Kotlin files
* No Compose

# Main Screen

Use:

* DrawerLayout
* NavigationView (left hamburger menu)
* AppBar/Toolbar

Main content contains:

* TabLayout
* ViewPager2

Tabs:

### Tab 1: Count Down

Features:

* Time input fields:

  * Hours
  * Minutes
  * Seconds
* Running timer display
* Start button
* Pause button
* Resume button
* Reset button
* "Speak Aloud" checkbox
* Speaking Interval dropdown (Spinner)

Speaking interval values:

* Every 1 second
* Every 5 seconds
* Every 10 seconds
* Every 15 seconds
* Every 30 seconds
* Every 60 seconds

Behavior:

* Countdown decreases each second.
* When Speak Aloud is enabled:

  * Use Android TextToSpeech.
  * Speak remaining time according to selected interval.
* At completion:

  * Speak:
    "Timer finished"
  * Show notification.
  * Show alert dialog.

---

### Tab 2: Count Up

Features:

* Running timer display
* Start button
* Pause button
* Resume button
* Reset button
* "Speak Aloud" checkbox
* Speaking Interval dropdown

Behavior:

* Counts upward from 00:00:00.
* Uses same TextToSpeech logic.
* Speaks elapsed time according to selected interval.
